 # resource-sharing-project
项目介绍： 积分制的资源共享平台项目的介绍，该项目平台提供八大交互系统，包括推荐系统、消息私聊系统、兑换系统、搜索系统、关注系统、分享系统、足迹系统、评论评分系统。  
技术栈：Vue2+Vue-cli+Axios+Vuex+ElementUI+Vue-router+Node
